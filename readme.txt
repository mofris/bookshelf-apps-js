Saya melakukan improvement pada website tugas akhir saya, Di website yang sudah saya buat terdapat navbar disisi atas yang meliputi :
1. Dashboard    : menampilkan semua data buku
2. Belum dibaca : untuk menampilkan data buku yang belum dibaca
3. Sudah dibaca : untuk menampilkan data buku yang sudah dibaca
4. login        : untuk keluar dari aplikasi

saya menggunakan teknik DOM Manipulatif yang sudah dipelajari pada materi sebelumnya. saya berinisiatif melakukan Manipulatif seperti
jika navbar 'belum dibaca' diclick, menampilkan data semua buku yang belum dibaca pada halaman dashboard.html
ini awalnya card belum dibaca saya beri class 'belum-dibaca-disactive' yang memiliki properti css display : none. ketika button belum dibaca diclick maka class pada card tersebut berubah menjadi 'belum-dibaca-active' yang menghapus properti css display : none tersebut, sehingga tampilan data belum dibaca tampil ke layar.
